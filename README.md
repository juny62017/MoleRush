# MoleRush

MoleRush is a fun whack a mole style game made by me 
where out of 9 holes you will have to click on the creature whenever it comes out
then you will 1 point ... 

## Features

- Random mole spawning
- each valid click will give you 1 point 
- Countdown timer of 30 seconds
- Game over after the timer ends
- Restart button
- Best score tracker
- Difficulty increase system
- Smooth animations


## Controls

- Click on the mole to score points

## Tech 

- HTML
- CSS
- Java

## How To Play

1. Click the Start Game button
2. Moles will randomly come out of holes
3. Click the mole quickly to score points
4. Each successful hit increases your score
5. The game speed increases over time
6. Try to score as much as possible before the timer ends
